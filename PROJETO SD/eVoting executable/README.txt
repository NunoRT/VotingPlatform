Estão presentes nesta pasta 4 ficheiros .jar

Por favor, corra cada um desses ficheiros no terminal com o comando "java -jar file_name.jar"
Nota: O ficheiro ServerRMI.jar deve ser corrido primeiro para ativar o servidor de dados.


IMPORTANTE!
- Verifique se o ficheiro configurations.properties se encontra dentro da pasta que tem os outros ficheiros .jar
- Por favor, abra o ficheiro configurations.properties e mude a informação de acordo com a descrita abaixo: 

UDP_PORT = Um valor 1024 a 49151
LOOKUP = electionsUC
REGISTRYPORT = Um valor de 1024 a 49151
REGISTRYIP = RMI server machine's IP
